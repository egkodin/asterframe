# Asterframe brand assets

This directory contains the canonical repository branding:

- [`banner.svg`](banner.svg) — wide banner used at the top of the repository README.
- [`icon.svg`](icon.svg) — square project icon for avatars, documentation, and integrations.

GitHub repositories do not expose a native per-repository icon in the standard repository header. Use `icon.svg` wherever a square project mark is supported, and upload a rasterized version manually when a service requires PNG or JPEG.
